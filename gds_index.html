Projects

Medical Data Mining for Disease Prediction and Personalized Treatment

Project Dates: January 2024 - present

Research Areas: Medical data mining; Disease prediction; Personalized treatment; Machine learning algorithms; Data visualization

Project Description: This project focuses on using data mining and machine learning to analyze large medical datasets. It aims to predict disease progression, identify risk factors, and provide personalized treatment recommendations by integrating diverse medical data and visualizing findings for healthcare providers. The goal is to enhance medical decision - making and advance precision medicine.
